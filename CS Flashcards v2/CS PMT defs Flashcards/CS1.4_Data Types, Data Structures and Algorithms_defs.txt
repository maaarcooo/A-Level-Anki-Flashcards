What is the AND (∧) logical operator? | A logical operator which returns TRUE (or 1) if and only if all inputs are TRUE (or 1)
Which logical operator returns TRUE only when all inputs are TRUE? | AND (∧)
What is ASCII? | A character set used to represent alphanumeric characters or symbols as a set of 8 bits
What character set uses 8 bits to represent alphanumeric characters or symbols? | ASCII
What is binary? | A number system that only uses ones and zeros to represent numbers (a base 2 system)
What base system is binary? | Base 2
What is bitwise manipulation? | Operations performed on a set of bits
What is a Boolean data type? | A data type that can only store one of two possible values (1 or 0, TRUE or FALSE)
Which data type can only store two possible values such as TRUE or FALSE? | Boolean
What is a Character data type? | A data type for storing a letter, number or special character
What is denary? | A number system that uses 10 characters (0 to 9) to represent numbers (a base 10 system)
What base system is denary? | Base 10
What is floating point arithmetic? | Performing arithmetic operations on floating point numbers in binary
What is hexadecimal? | A number system that uses 16 characters (0 to 9 and A to F) to represent numbers (a base 16 system)
What base system is hexadecimal? | Base 16
What characters are used in hexadecimal? | 0 to 9 and A to F
What is an Integer data type? | A data type for storing whole number values (positive or negative) with no decimal parts
Which data type stores whole numbers with no decimal parts? | Integer
What is the OR (∨) logical operator? | A logical operator which returns TRUE (or 1) if and only if any one of the inputs are TRUE (or 1)
Which logical operator returns TRUE when at least one input is TRUE? | OR (∨)
What is a primitive data type? | A basic built-in data type provided by a programming language
What is a Real/Floating Point data type? | A data type for storing numbers with decimal or fractional parts
Which data type stores numbers with decimal or fractional parts? | Real/Floating Point
What are shifts in bitwise manipulation? | A bitwise manipulation where all bits are moved by one place in a given direction, with the end bit taken as a carry or appended to the other end depending on the shift method
What is a String data type? | A data type for storing a sequence of alphanumeric characters or symbols, typically within quotation marks
Which data type stores a sequence of characters typically within quotation marks? | String
What is two's complement? | A method of storing negative numbers in binary by flipping all bits of the positive number's binary representation and then adding 1
How do you convert a positive binary number to its negative using two's complement? | Flip all the bits and add 1
What is UNICODE? | A character set that is a superset of ASCII, representing alphanumeric characters or symbols as an integer code point equal to that character's ASCII code
How does UNICODE relate to ASCII? | UNICODE is a superset of ASCII
What is the XOR logical operator? | A logical operator which returns TRUE (or 1) if and only if exactly 1 of the inputs are TRUE (or 1)
Which logical operator returns TRUE only when exactly one input is TRUE? | XOR
What is an array? | A data structure for storing a finite, ordered set of data of the same data type within a single identifier
What are the key characteristics of an array? | Finite, ordered, same data type, single identifier
What is a binary search tree? | A tree where each node cannot have more than 2 children, with the right node and descendants always having a greater value than the root node
How many children can each node have in a binary search tree? | No more than 2
What is breadth first traversal? | A method of traversing a graph by visiting all neighbours of the first node before repeating with each neighbour in the order they were visited
What is depth first traversal? | A method of traversing a graph by travelling as far as possible along one route before backtracking and trying alternative unexplored routes
What is a directed graph? | A graph where the order of vertices paired in an edge matters; edges are one way
What is the difference between directed and undirected graphs? | In directed graphs, edges are one way; in undirected graphs, edges are bidirectional
What is a graph data structure? | A data structure consisting of a set of vertices/nodes connected by a set of edges/arcs
What is a hash table? | A data structure where a hashing algorithm calculates a value to determine where a data item is stored, allowing direct access by recalculation without search
How does a hash table allow direct access to data? | By recalculating the hash value, no search is required
What is a linked list? | A data structure storing an ordered sequence of data where each item is stored with a pointer to the next item, not stored contiguously in memory
How are items connected in a linked list? | Each item is stored with a pointer to the next item
Are linked list items stored contiguously in memory? | No, they are not stored contiguously
What is a list data structure? | A data structure that stores a sequence of data values, each with a unique index
What is a queue? | A first-in-first-out (FIFO) data structure where the first item added is the first to be removed
What does FIFO stand for and which data structure uses it? | First-In-First-Out; Queue
What is a record? | A data structure that stores data in elements called fields, organised based on attributes
What are the elements in a record called? | Fields
What is a stack? | A last-in-first-out (LIFO) data structure where the last item added is the first to be removed
What does LIFO stand for and which data structure uses it? | Last-In-First-Out; Stack
What is a tree data structure? | A data structure using a set of linked nodes to form a hierarchical structure starting at a root node, where each node is a child of a parent node
What is a tuple? | A data structure for storing an immutable, ordered set of data which can be of different data types, within a single identifier
What makes tuples different from arrays? | Tuples are immutable (cannot be modified once created) and can contain different data types
What is an undirected graph? | A graph where the order of vertices paired in an edge does not matter; edges are bidirectional
What are the association laws in Boolean algebra? | A ∧ (B∧C) = (A∧B) ∧ C and A ∨ (B∨C) = (A∨B) ∨ C
What is a Boolean expression? | A combination of Boolean variables and logical operators which evaluates to either TRUE or FALSE depending on the input
What is Boolean logic? | A type of algebra with logical operators where all values and expressions ultimately reduce to TRUE or FALSE
What are the commutation laws in Boolean algebra? | A∧B = B∧A and A∨B = B∨A
What is De Morgan's First Law? | ¬(A∨B) = ¬A ∧ ¬B
What is De Morgan's Second Law? | ¬(A∧B) = ¬A ∨ ¬B
What is the distribution law for AND over OR? | A∧(B∨C) = (A∧B) ∨ (A∧C)
What is the double negation law? | A = ¬¬A
What is a D-Type flip flop? | A sequential logic circuit used to store a single bit with two stable states that can be flipped between using an input signal
How many bits can a D-Type flip flop store? | A single bit (1 bit)
What is a full adder? | A combination of two half adders that takes a carry bit and two other input bits and returns their sum and new carry as two output bits
How many input bits does a full adder take? | Three (a carry bit and two other input bits)
What is a half adder? | A combinational arithmetic circuit that adds two numbers and produces a sum bit (S) and carry bit (C) as output
What outputs does a half adder produce? | A sum bit (S) and a carry bit (C)
What is a Karnaugh map? | A method of simplifying Boolean expressions by redrawing the truth table and applying visual rules to obtain expressions with minimum logical operators as a sum of products
What is the purpose of a Karnaugh map? | To simplify Boolean expressions to obtain minimum logical operators
What is a logic gate diagram? | A graphical method of representing Boolean expressions using standard symbols for logic gates
