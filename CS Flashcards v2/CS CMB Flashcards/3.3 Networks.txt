What is a network? | Two or more interconnected devices designed to share resources, exchange data, and communicate with each other
What are the three main purposes of a network? | To enable data sharing, resource sharing, communication, and collaboration
What is a protocol in networking? | A set of rules defining how two computers communicate with each other
Why are protocols standard across all devices? | So that all devices have a designated method of communicating with each other, regardless of manufacturer
A set of rules defining how two computers communicate is called what? | A protocol
What does HTTP stand for? | HyperText Transfer Protocol
What is HTTP used for? | Transferring web content (text, images, video) and web page rendering
What is HTTPS? | HTTP Secure - a version of HTTP with encryption for security
What is HTTPS used for? | Secure transactions like online banking and shopping
What does TCP/IP stand for? | Transmission Control Protocol / Internet Protocol
What is TCP/IP used for? | Routing packets through networks; it's the main suite of protocols used for internet communication
What does POP3 stand for? | Post Office Protocol (version 3)
What does IMAP stand for? | Internet Message Access Protocol
What are POP3 and IMAP used for? | Email access - mailing protocols
What does FTP stand for? | File Transfer Protocol
What is FTP used for? | Transmission of files over networks, with authentication and file directory management
What does SMTP stand for? | Simple Mail Transfer Protocol
What is SMTP used for? | Sending email messages between servers
What does ARP stand for? | Address Resolution Protocol
What is ARP used for? | Translating IP addresses into MAC addresses to ensure packets reach the correct device
What does UDP stand for? | User Datagram Protocol
How does UDP differ from TCP? | UDP is a simpler, connectionless protocol that doesn't guarantee delivery, disregards order and error-checking, making it faster
What does TCP provide that UDP does not? | Reliable, ordered, and error-checked delivery of packets
What is a standard in networking? | A set of guidelines or frameworks that govern how a task should be performed or how a product should function
Why are standards needed in networks? | To ensure compatibility, interoperability, and consistency across different devices and software
What do standards enable in terms of network devices? | Different network devices, regardless of manufacturer or model, to work together seamlessly
How do standards support network growth? | They support network expansion and integration of new technologies without disrupting existing operations
What is protocol layering? | The way network protocols are divided into layers, each performing specific functions
What are the benefits of protocol layering? | Modular design, simplified troubleshooting, promotes interoperability, and allows layers to be altered independently
What is the TCP/IP stack? | A stack of networking protocols that work together passing packets during communication
How many layers are in the TCP/IP model? | Four layers
Name the four layers of the TCP/IP stack from top to bottom | Application Layer, Transport Layer, Internet Layer (Network Layer), Link Layer
Which layer is at the top of the TCP/IP stack? | Application Layer
What does the Application Layer do? | Specifies what protocol needs to be used to relate the application being sent (e.g., HTTP, POP3, FTP)
Which layer uses TCP to establish end-to-end connections? | Transport Layer
What does the Transport Layer do to data? | Splits data into packets and labels them with packet number, total number of packets, and port number
What happens if packets get lost during transmission? | The Transport Layer requests retransmissions of the lost packets
What does the Internet Layer (Network Layer) add to packets? | Source and destination IP addresses
What is a socket address? | The combination of an IP address and a port number
Which network device operates on the Network Layer? | Routers
What does the Link Layer add to packets? | MAC addresses identifying the Network Interface Cards of source and destination computers
What is another name for the Link Layer? | Network Interface Layer
What does the Link Layer do to digital packets for transmission? | Translates them into electrical, optical, or wireless signals
On the receiving computer, in what order are TCP/IP layers processed? | Bottom to top (Link, Network, Transport, Application)
What does the Transport Layer do when receiving data? | Removes port number and reassembles packets
What does the Application Layer do when receiving data? | Presents the data to the recipient in the form it was requested
What is encapsulation in networking? | Converting data into a format that can be sent and received over the network, adding headers at each layer
What are packets? | Units of data that are small and easier to manage; segments of data being transmitted
What three parts does a packet contain? | Header, Payload, and Trailer (Footer)
What information is contained in a packet header? | Sender and recipient IP addresses, protocol being used, order of packets, Time To Live (TTL)
What is the payload of a packet? | The raw data to be transmitted
What does the trailer of a packet contain? | Checksum or cyclic redundancy check for error detection
What is Time To Live (TTL) in a packet? | A value that tells the packet when to expire so it does not travel forever
What is another name for Time To Live? | Hop Limit
What is the purpose of IP addresses in packet headers? | To allow packets to be delivered to the correct destination and enable tracing of the source
Why is the protocol included in the packet header? | To allow the recipient computer to understand how to interpret the packet
What is the purpose of packet order information in the header? | To allow packets to be reconstructed in the appropriate order at the recipient device
What is a checksum used for? | To detect whether any errors have occurred during transmission
What is a cyclic redundancy check? | A code used to detect transmission errors, found in the packet trailer
What is an IP address? | A unique identifier for a device on a network
What are the two versions of IP addresses in use? | IPv4 and IPv6
Give an example of an IPv4 address format | 104.22.74.202
Give an example of an IPv6 address format | 0000:0000:0000:0000:0000:ffff:6816:4aca
What does LAN stand for? | Local Area Network
What is a LAN? | A network spread over a small geographical area or a single site
What does WAN stand for? | Wide Area Network
What is a WAN? | A network spread over a large geographical area, connecting multiple LANs
Who typically owns the hardware in a LAN? | A single entity/organisation
Why are LANs considered more secure than WANs? | Because all the hardware is owned by the organisation, unlike WANs which use third-party connections
What type of connections do WANs typically use? | Third-party connections such as leased lines, fibre, or satellite links
What is the largest WAN? | The Internet
What is the Internet? | A network of networks allowing computers on opposite sides of the globe to communicate
How are continents connected to form the Internet? | Using large international backbone cables, many passing underwater
What extra hardware do WANs typically require compared to LANs? | Extra telecommunication hardware
Give an example of when a WAN would be used | A company connecting offices in multiple cities or countries
What is a disadvantage of WANs regarding speed? | Typically slower than LANs due to greater transmission distances
What is a disadvantage of WANs regarding cost? | More expensive to set up and maintain because of leased lines or ISP costs
What is a security disadvantage of WANs? | More vulnerable to security risks as data travels across public networks
What is the Domain Name System (DNS)? | A system used to name and organise internet resources; translates domain names into IP addresses
What is a DNS server's role? | To translate domain names into IP addresses when accessing a website
Why are domain names used instead of IP addresses? | Domain names are much easier to remember than IP addresses
What does TLD stand for in DNS? | Top Level Domain
What does 2LD stand for in DNS? | Second Level Domain
Give an example of DNS hierarchy | In 'leeds.gov.uk': uk is TLD, gov is 2LD, leeds is the subdomain
What is a DNS resolver? | The first stop in the DNS lookup process, usually provided by your ISP
What do root servers do in DNS? | Direct the resolver to find the top-level domain server (like .com, .org, .edu)
What do TLD servers store? | Information about domains under that top-level domain
What is an authoritative DNS server? | The server that responds with the actual IP address of the requested domain
What is DNS propagation? | When a domain is newly registered or changes IP address, the DNS record needs to be updated across DNS servers
What are the steps when you type a URL into a browser? | 1) Check local cache, 2) Query DNS resolver, 3) Query root server, 4) Query TLD server, 5) Query authoritative server, 6) Receive IP, 7) Request web page, 8) Receive and render page
What is packet switching? | A method of communication where data is broken into packets and sent across different routes, then reassembled at the destination
In packet switching, do all packets take the same route? | No, packets are sent across the most efficient route, which can vary for each packet
What is an advantage of packet switching regarding network failures? | If one path breaks, another can be used; only lost packets need to be resent
What is a disadvantage of packet switching? | Time is spent deconstructing and reconstructing packets; must wait for all packets before data can be received
What is circuit switching? | A method of communication where a dedicated direct link is created between two devices for the entire conversation
In circuit switching, what requirement exists for data transfer rates? | Both devices must transfer and receive data at the same rate
What is an advantage of circuit switching? | Data arrives in logical order, enabling quicker reconstruction; no delay in speech for calls
What is a disadvantage of circuit switching? | Bandwidth is wasted during periods when no data is being sent; ties up network sections
What type of data transmission is packet switching best suited for? | Data that can tolerate some delay, such as emails and web pages
What type of data transmission is circuit switching best suited for? | Real-time services like voice calls or video conferencing that require low latency
Why is circuit switching less efficient than packet switching? | Resources remain allocated during the whole conversation, even when no data is being sent
Why is packet switching more scalable than circuit switching? | It can accommodate large amounts of data and many users without dedicated paths
What is a firewall? | A device designed to prevent unauthorised access to a network
How does a firewall work? | It passes packets between two NICs and compares them against preconfigured rules (packet filters)
What is packet filtering (static filtering)? | Limiting network access by examining source IP, destination IP, protocols, and ports against administrator rules
What two things can happen when a firewall denies access? | The packet can be dropped (no notification) or rejected (alert sent to sender)
What is the difference between a dropped and rejected packet? | A rejected packet sends an alert to the sender; a dropped packet does not
What is a proxy server? | An intermediary that collects and sends data on behalf of the user
What are four benefits of using proxy servers? | User privacy/anonymity, caching frequently used website data, reducing overall web traffic, preventing access to certain websites
How can proxies be used by administrators? | To prevent access to sensitive or irrelevant information at work or school
What is encryption? | A way of keeping data secure by making it unreadable if intercepted during transmission
How is encrypted data made readable again? | Data is encrypted and decrypted using a set of keys
What is a Network Interface Card (NIC)? | A hardware component that enables a device to connect to a network
What does a NIC assign to each device? | A unique MAC address
What does MAC stand for? | Media Access Control
What is a MAC address? | A 48-bit value coded into the device, usually written as a twelve-digit hexadecimal number
How many bits is a MAC address? | 48 bits
How is a MAC address typically written? | As a twelve-digit hexadecimal number
What is a switch in networking? | A device used to direct the flow of data across a network to specific intended devices
How does a switch differ from a hub? | A switch only sends data to the intended device using a lookup table, while a hub broadcasts to all devices
What does a switch's lookup table contain? | Port numbers matched to MAC addresses
What happens when a switch receives a data packet? | It examines the destination MAC address, looks it up in its table, and forwards to the corresponding port
What is a hub in networking? | A networking device that connects multiple devices and broadcasts all received data to all connections
Why are hubs considered "dumb" devices? | They pass on anything received to all other connections without examining the destination
What are the disadvantages of using hubs? | Creates unnecessary network traffic and security concerns as all devices receive all data
What is a Wireless Access Point (WAP)? | A device that allows devices to connect to a network wirelessly
How does a WAP connect to the network? | From a fixed location using Ethernet or fibre optic cable
What is a router? | A device used to connect two or more networks together
What is the primary function of a router? | To analyse data packets and determine the best path for packets to reach their destination
What does a router use to forward packets? | A routing table of known networks
What is a hop in networking? | Each pass from router to router as a packet travels to its destination
What additional functions can routers perform? | Assigning IP addresses to devices, filtering incoming traffic, wireless networking, firewall functions
What is a modem? | A device that modulates analogue signals to encode digital information and demodulates to decode transmitted information
What does modem stand for? | Modulator-demodulator
What does a modem enable? | A computer to transmit data over telephone or cable lines by converting digital signals to analogue and vice versa
What is a gateway in networking? | A device used when protocols are not the same between networks; it translates protocols
How does a gateway work? | By removing the header from packets and adding data to packets using the new protocol
What are cables used for in networking? | Physical paths for data to travel between devices
Name three types of Ethernet cables | Cat5, Cat5e, Cat6
What do fibre-optic cables use to transmit data? | Light
What advantage do fibre-optic cables have over copper cables? | Much higher speed and larger data capacity
What is a client-server network? | A network where powerful server computers control the network and serve clients with services
In a client-server network, where are resources managed? | Centrally on the server
What is an advantage of client-server networks regarding management? | Easier central management with central backups
What is an advantage of client-server networks regarding scalability? | New clients can be added easily
What is a disadvantage of client-server networks? | Single point of failure - if the server goes down, services are unavailable
What is a disadvantage of client-server networks regarding cost? | Expensive to set up and maintain; often needs dedicated maintenance teams
What is a peer-to-peer network? | A network where all computers share equal responsibility with no central server
In a peer-to-peer network, what status do all machines have? | Equal status - each acts as both server and client
What is an advantage of peer-to-peer networks? | Easy to set up, less expensive, no dependency on a central server
What is a disadvantage of peer-to-peer networks? | Lack of central control can lead to security issues; not suitable for large networks
Who is responsible for security and backups in a peer-to-peer network? | Each individual machine's user
Why are peer-to-peer networks used in piracy? | It's almost impossible to trace the origin of files
What type of organisation would typically use a client-server network? | Larger organisations where centralised control, reliability, and security are paramount
What type of network would typically use peer-to-peer? | Home networks, small businesses, or specific applications like file sharing
What is a virus in computer security? | Malicious software that attaches to legitimate programs, replicates to spread, and can damage systems
What is malware? | Malicious software designed to harm or gain unauthorised access to a system or network
What is a worm? | A standalone malicious program that can spread and replicate itself over computer networks
How does a worm differ from a virus? | A worm is standalone and spreads over networks; a virus attaches to programs and requires user action to spread
What is a Trojan horse? | A program that disguises itself as legitimate but can delete data or damage hardware when installed
What is spyware? | Software that records all key presses and transmits them to a third party
What is adware? | Software that displays unwanted advertisements without user consent
What is ransomware? | Malware that encrypts user files and demands ransom payment to decrypt them
What is a Denial of Service (DoS) attack? | An attack where a computer floods a server with requests, causing it to crash or become unavailable
What is a DDoS attack? | A Distributed Denial of Service attack using multiple computers as bots to flood a server
What is SQL injection? | An attack technique where malicious SQL statements are inserted into entry fields to exploit security vulnerabilities
What is phishing? | Attempting to acquire sensitive information by masquerading as a trustworthy entity via email with fake website links
What is pharming? | A cyber attack using malware to redirect website traffic to a bogus site without user knowledge
How does pharming differ from phishing? | Pharming uses malware to redirect automatically; phishing requires user to click a link
What is social engineering? | Manipulating individuals to gain access to confidential information or perform actions benefiting the attacker
Give examples of social engineering techniques | Posing as someone else, enticing with desirable items, leaving infected USB drives, pretending to be bank representatives
What is hacking? | Gaining unauthorised access to a system or network to steal/manipulate data, disrupt services, or cause damage
What are secure passwords? | Strong, complex, and unique passwords that help protect against unauthorised access
What is anti-virus software? | Programs designed to detect and neutralise or remove malicious software
What is anti-spyware software? | Tools to detect and remove spyware and other kinds of malware
What is two-factor authentication (2FA)? | Security requiring users to provide two forms of identification, usually a password and a code sent to phone/email
Why are regular software updates important for security? | They ensure systems have the latest security patches
What is packetising? | The process of dividing a large message into smaller, manageable units called packets
What information does a packet header typically contain? | Source IP, destination IP, sequence number, protocol, packet length, checksum
What is the sequence number in a packet header used for? | Helps in reassembling packets back into the original message at the receiving end
What happens during packet reassembly? | Packets are reassembled back into the original message using information in the headers
What is network congestion a potential issue for? | Packet switching - can lead to packet loss
Why might packets arrive out of order? | In packet switching, packets can take different routes to the destination
What is the role of the Internet Layer in receiving data? | Removes IP addresses from packets
What protocols operate at the Application Layer? | HTTP, HTTPS, FTP, SMTP, POP3, IMAP
Which TCP/IP layer is responsible for routing? | Internet Layer (Network Layer)
What does the switch use MAC addresses for? | To identify which port to forward data packets to
What makes client-server networks more secure than peer-to-peer? | Data is stored in one central location with centralised security management
Why might resources be difficult to locate in peer-to-peer networks? | Data is spread around the network with no central storage location
What is the main trade-off between packet and circuit switching? | Packet switching is more efficient but may have delays; circuit switching guarantees real-time delivery but wastes bandwidth
