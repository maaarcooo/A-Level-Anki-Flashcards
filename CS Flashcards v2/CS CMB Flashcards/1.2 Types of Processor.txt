What does RISC stand for? | Reduced Instruction Set Computer
What does CISC stand for? | Complex Instruction Set Computer
What type of processor has a small instruction set with simple instructions? | RISC (Reduced Instruction Set Computer)
What type of processor has a large instruction set with complex instructions? | CISC (Complex Instruction Set Computer)
How many clock cycles does each RISC instruction take to execute? | One clock cycle
How many clock cycles can CISC instructions take to execute? | More than one clock cycle (multiple clock cycles)
Why is RISC more suitable for pipelining? | Each instruction takes one clock cycle to execute
Why is CISC not well suited to pipelining? | Instructions take multiple clock cycles to execute
Which processor type requires more work from the compiler to translate high-level code? | RISC processors
Which processor type requires less work from the compiler? | CISC processors
Which processor type requires more RAM to store code? | RISC (because code is longer with more instructions)
Which processor type requires less RAM to store code? | CISC (because code is shorter with fewer instructions)
Which processor type has fewer transistors? | RISC
Which processor type has more transistors? | CISC
Which processor type has fewer general purpose registers? | RISC
Which processor type has more general purpose registers? | CISC
Which processor type has fewer addressing modes? | RISC
Which processor type has more addressing modes? | CISC
Which processor type requires less power? | RISC
Which processor type requires more power? | CISC
Which processor type costs less to manufacture? | RISC
Which processor type costs more to manufacture? | CISC
Where are RISC processors typically used? | Smartphones and tablets
Where are CISC processors typically used? | Laptops and desktop computers
Where are CISC processors commonly used in embedded applications? | Microcontrollers and embedded systems
Can a program written for a RISC processor run on a CISC processor? | No, programs are not compatible between RISC and CISC
Can a program written for one RISC processor necessarily work on another RISC processor? | No, they may have different instruction sets
What is the aim of CISC architecture regarding assembly code? | To accomplish tasks in as few lines of assembly code as possible
Where are CISC instructions built into? | The hardware
What is a GPU? | Graphics Processing Unit - a processor with many independent cores that work in parallel for repetitive tasks
What type of co-processor is a GPU? | A secondary processor designed to supplement the activities of the primary processor
What makes GPUs efficient at completing repetitive tasks? | Lots of independent processors that work in parallel
What is the primary purpose of a GPU in a computer? | Processing graphics to reduce the load on the CPU
How are CPUs and GPUs different in terms of purpose? | CPUs are general purpose processors; GPUs are designed specifically for graphics
What can GPUs perform on multiple pieces of data at one time? | An instruction (SIMD processing)
Where can a GPU be located? | Part of the graphics card or embedded in the CPU
How many processing cores do modern GPUs typically contain? | Hundreds or even thousands of smaller processing cores
What type of processing do GPU cores perform? | Parallel processing
Give two examples of tasks GPUs are efficient at completing. | Image processing and machine learning
What is 3D modelling used for with GPUs? | Rendering lighting effects, textures and shadows
How do GPUs help with data modelling? | They can handle many calculations simultaneously for large datasets and complex operations like sorting and filtering
How are GPUs used in financial modelling? | Simulating different scenarios in risk modelling, option pricing; running many simulations in parallel
What is data mining? | The process of analysing large amounts of data to find patterns
What computational tasks are involved in data mining? | Sorting, searching, pattern recognition, statistical analysis and graph algorithms
What complex numerical calculations can GPUs perform in parallel? | Matrix multiplication and inversion
What types of simulations benefit from GPU parallel processing? | Physics and engineering simulations involving complex maths models
How are GPUs used in machine learning training? | Training on massive amounts of data in parallel with matrix multiplications and other computations
How are GPUs used after machine learning training? | Speeding up the process of making predictions on new data
What does SIMD stand for? | Single Instruction Multiple Data
What is SIMD processing? | Computers with multiple processing elements that perform the same operation on multiple data points simultaneously
Why do GPUs support SIMD processing? | They were originally designed to perform the same operations on multiple pixels or vertices simultaneously
What specialist instructions are GPUs designed to execute? | Operations on matrices, vectors and geometric transformations common in 3D graphics rendering
How are CPU cores different from GPU cores in terms of processing? | CPU cores are optimised for serial processing; GPU cores are optimised for parallel processing
What is a benefit of GPU parallel processing? | GPUs can handle many tasks simultaneously as they are multicore processors
Why are GPUs considered more energy efficient than CPUs for parallel tasks? | They can perform more calculations per unit of power consumed
What is parallel processing? | When a computer has multiple cores that can work on tasks simultaneously
In parallel processing, what can each core do? | Work on the same task to complete it more quickly, or work on separate tasks at the same time
What is a multicore system? | A system with more than one processing unit in a single processor that can independently process instructions at the same time
How can parallel processing be achieved besides multicore CPUs? | By utilising more than one processor (e.g., a CPU and a GPU)
What is threading in the context of parallel systems? | A method that allows parallel systems to complete tasks with a single core
Which generally performs better in larger projects: multicore or parallel systems? | Multicore systems
What is a speed benefit of parallel processing? | If a task can be divided into subtasks executed simultaneously, total execution time can be reduced
What is Amdahl's Law limitation in parallel processing? | Even with infinite processors, there is a limit to maximum speed improvement if part of the program can't be parallelised
Why is parallel programming more complex than serial programming? | Tasks have to be synchronised and data shared correctly
Why is debugging parallel programs more difficult? | Due to the precise timing of specific events
What overhead can reduce parallel processing benefits? | Communication between processors can take significant time and resources
What is a limited applicability drawback of parallel processing? | Not all tasks can be run in parallel and must be executed serially
What is the multitasking benefit of multicore processors? | Each core can work on a different task, effective when multiple applications are open
How do multicore processors help with background tasks? | They can assign background tasks like anti-malware scans to one core, reducing impact on other tasks
How do multicore processors improve responsiveness? | If a program becomes unresponsive, other cores continue running their tasks
What types of applications benefit significantly from parallel processing? | Real-time applications including graphics rendering
What is improved performance benefit of parallel processing for data? | Simultaneous computation can take place on different data subsets
Where is simultaneous data computation used? | Machine learning, data mining and scientific computing
What is better resource utilisation in parallel processing? | Multi-core or multiple processors can be used more effectively
What types of problems benefit from parallel processing? | Large and complex problems that can be divided into parallel subtasks
A processor with a small instruction set where each instruction takes one clock cycle is called what? | RISC (Reduced Instruction Set Computer)
A processor with a large instruction set where complex instructions can take multiple clock cycles is called what? | CISC (Complex Instruction Set Computer)
A secondary processor designed to supplement the primary processor is called what? | A co-processor
A processor with many independent cores optimised for parallel processing of repetitive tasks is called what? | GPU (Graphics Processing Unit)
Multiple processing units in a single processor that independently process instructions simultaneously describes what? | A multicore system
Computers with multiple processing elements performing the same operation on multiple data points simultaneously describes what? | SIMD (Single Instruction Multiple Data) processing
