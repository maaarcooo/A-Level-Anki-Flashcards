What is HTML? | A language/script that web pages are written in, which allows a browser to interpret and render a webpage by describing its structure and order
What does HTML stand for? | HyperText Markup Language
What is the purpose of HTML? | To describe the structure and order of a webpage so a browser can interpret and render it
What notation does HTML use for its elements? | Tags written in angle brackets (e.g., <tag>, </tag>)
What are the two main sections of an HTML webpage? | The head and the body
What does the head section of an HTML document contain? | The title of the webpage
What does the body section of an HTML document contain? | The content of the webpage
What does the <html> tag do? | Indicates that all code within these tags is interpreted as HTML
What does the <body> tag do? | Defines the content in the main content area of the webpage
What does the <link> tag do? | Used to link to additional files, including CSS stylesheets
What does the <head> tag do? | Defines the browser tab or window heading area
What does the <title> tag do? | Defines the text that appears in the browser tab or window heading area
What do <h1>, <h2>, <h3> tags represent? | Heading styles in decreasing sizes
What does the <p> tag do? | Defines a paragraph separated with a line space above and below
What does the <img> tag do? | Used to display images on a webpage
What type of tag is <img>? | A self-closing tag (no closing tag required)
What parameters must be included with the <img> tag? | src (source), height, and width
What does the <a> tag do? | Creates an anchor/hyperlink to another location
What is the syntax for creating a hyperlink in HTML? | <a href="location">link text</a>
What does the href attribute specify in an anchor tag? | The URL/location the hyperlink points to
What does the <ol> tag do? | Defines an ordered (numbered) list
What does the <ul> tag do? | Defines an unordered (bulleted) list
What does the <li> tag do? | Defines each element/item within a list
What does the <div> tag do? | Divides a page into separate areas that can be referred to uniquely by name
How do you create a named division in HTML? | <div id="name">
What are classes and identifiers in HTML? | Attributes given to elements on a webpage which you wish to style in a particular way
What is the difference between a class and an identifier in HTML/CSS? | A class can be used by multiple elements, while an identifier is unique to only one element
How are CSS classes defined? | Using a full stop/period (e.g., .classname { })
How are CSS identifiers defined? | Using a hashtag (e.g., #identifiername { })
How do you apply a class to an HTML element? | Using the class attribute (e.g., <div class="example">)
How do you apply an identifier to an HTML element? | Using the id attribute (e.g., <div id="example">)
What is CSS? | Cascading Style Sheets - a language used to describe the style of a webpage
What is the purpose of CSS? | To specify the way HTML elements look and apply styling to tags
What are the two forms CSS can be used in? | Internal/embedded CSS and external CSS
What is internal/embedded CSS? | CSS placed inside style tags and entered directly into the HTML document
What is external CSS? | CSS written in a separate document with a link added to the HTML document
What is the syntax for linking an external CSS stylesheet? | <link href="filename.css" rel="stylesheet" type="text/css">
Where should the link to an external CSS stylesheet be placed? | In the header section of the HTML document
What is the basic syntax structure of CSS? | Selector name followed by curly brackets containing attribute definitions (e.g., selector { property: value; })
What is JavaScript? | A scripting language which primarily exists to add interactivity to websites
Is JavaScript interpreted or compiled? | Interpreted
Why is JavaScript being interpreted beneficial for web pages? | Web pages can be interpreted by the browser each time they are loaded, independently of processor architecture
What is a common use of JavaScript? | Validating input data on the client computer
Name three advantages of using JavaScript for input validation | 1) Local computer handles invalid data before sending to servers 2) Eases load on busy servers 3) Reduces web traffic
How do you select an HTML element by ID in JavaScript? | document.getElementById("elementId")
How do you change the content of an HTML element using JavaScript? | element.innerHTML = "new content"
How do you write directly to an HTML document using JavaScript? | document.write("content")
How do you display an alert box in JavaScript? | alert("message")
What is a search engine? | A program that searches through a database of internet addresses looking for resources based on criteria set by the client
Why is the order of search results important? | It determines which websites users choose to visit, so high-quality relevant content should appear at the top
What do search engines rely on to return results? | An index of web pages
What are web crawlers? | Programs used to collect information about websites to build the search engine index (also called spiders)
What is another name for web crawlers? | Spiders
How do web crawlers work? | They traverse the Internet page by page using links, collecting keywords, phrases, and metadata from websites
What information do web crawlers collect from websites? | Keywords, phrases from linked pages, and metadata specified by website owners
What is metadata in the context of web crawlers? | Information specified by the website owner that describes the website
What does the PageRank algorithm do? | Ranks web pages to determine the order in which they are displayed in search results
What two factors determine the PageRank of a webpage? | 1) How many incoming links it has from other web pages 2) The page rank of the web pages that link to it
What data structure is used to represent web page links in PageRank? | A directed graph
In the PageRank graph structure, what do nodes represent? | Web pages
In the PageRank graph structure, what do arcs represent? | Links between web pages

In the PageRank formula, what does d represent? | The damping factor
What is the typical value of the damping factor (d) in PageRank? | 0.85
In the PageRank formula, what does Count(Tn) represent? | The total count of outbound links from webpage n
What initial value is assumed for unknown PageRank values in the first iteration? | 1
Why is the PageRank algorithm iterative? | Each iteration refines the values using results from the previous iteration until values converge
What additional factors do modern search engines consider beyond basic PageRank? | Age of a web page and how quickly content loads
What is server side processing? | When a client sends data to a server for it to be processed, with no processing on the client computer
Name two common server side scripting languages | SQL and PHP
What are the advantages of server side processing? | Does not require plugins, can perform large calculations faster, not browser dependent, more secure
What is client side processing? | When a client processes data on a local device, with all information processed on the client computer
What is another name for client side processing? | Client side scripting
What language is commonly used for client side scripting? | JavaScript
What are the advantages of client side processing? | Webpage can immediately respond to user actions, executes quickly, gives developers more control over website behaviour and appearance
Why is server side processing more secure than client side? | Processing happens on the server, so sensitive code and data are not exposed to the client
Why can server side processing perform large calculations faster? | Servers typically have more processing power than client computers
Why does client side processing execute quickly? | No need to send data to a server and wait for a response
